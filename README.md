# Verafide Button

### Install

```
npm i
npm run build
```

### Use in project

```
cp dist/verafide-button.js /myApp/public/

<script type="module" src="verafide-button.js"></script>

<verafide-button appKey="myappkey" />
<verafide-button appKey="otherappkey" hostname="http://localhost:3000" />
```
